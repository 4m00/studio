# **App Name**: CostVision

## Core Features:

- Executive Dashboard: Provides top-level analytics on operational costs, including KPI cards, cost dynamics graphs, and budget deviation alerts.
- Request List Management: Allows users to view, filter, and manage expense requests with detailed filtering options and bulk actions.
- Automated anomaly flagging: AI tool to intelligently flag unusual or unexpected data points.
- Expense Request Creation/Editing: Enables users to create and edit expense requests through a multi-step form with detailed expense itemization and budget control.
- Approval Workflow: Facilitates the expense request approval process with detailed request information and clear approval/rejection actions.
- Cost Analytics: Offers interactive cost analysis with customizable filters and visualizations to break down expenses by department and category.
- ML-Based Anomaly Detection: Uses machine learning to identify and analyze cost anomalies, providing AI-driven insights and recommendations.

## Style Guidelines:

- Primary color: Dark blue (#1e3a8a) for a corporate and trustworthy feel.
- Background color: Light gray (#f9fafb) for a clean and modern look.
- Accent color: Green (#10b981) to highlight success metrics and approvals, contrasting effectively with the dark blue primary color.
- Body and headline font: 'Inter', a sans-serif font for a modern and neutral aesthetic, ensuring readability across the application. Note: currently only Google Fonts are supported.
- Use Lucide Icons (outline style) for a consistent and modern look throughout the application.
- Employ modern cards with shadows and rounded corners (8px) to organize information and enhance user experience.
- Incorporate subtle hover effects to improve interactivity and provide feedback on user actions.
